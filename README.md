# Проект "Парсинг вакансий с hh.ru" 🔍

Это консольное приложение для поиска и управления вакансиями с платформы hh.ru.
Программа позволяет искать вакансии по ключевым словам, сохранять их в JSON-файл, а также фильтровать, сортировать и просматривать сохраненные данные.

## Основные возможности

* **Поиск вакансий** по ключевому слову с использованием API hh.ru.
* **Сохранение и управление** данными в JSON-файле.
* **Просмотр** всех сохраненных вакансий.
* **Фильтрация** вакансий по ключевым словам в описании.
* **Фильтрация** вакансий по диапазону зарплат.
* **Сортировка** и получение топ-N вакансий по зарплате.
* **Удаление** отдельных вакансий или полная очистка файла.

## Структура проекта

* `main.py`: Точка входа в программу. Здесь находится пользовательский интерфейс.
* `src/`: Директория с исходным кодом.
    * `api_classes.py`: Содержит классы для работы с API hh.ru.
    * `vacancy_class.py`: Класс для представления данных о вакансии.
    * `file_classes.py`: Классы для сохранения и чтения данных из JSON-файла.
    * `utils.py`: Вспомогательные функции, такие как фильтрация и сортировка.
* `data/`: Директория для хранения файла `vacancies.json`.
* `tests/`: Директория с тестами для проверки функционала.

## Как запустить проект

1.  **Клонируйте репозиторий:**
    ```bash
    git clone [https://github.com/new](https://github.com/new)
    ```

2.  **Перейдите в директорию проекта:**
    ```bash
    cd hh_project
    ```

3.  **Установите зависимости (если они есть, например, requests):**
    ```bash
    pip install -r requirements.txt
    # или, если используешь poetry:
    poetry install
    ```

4.  **Запустите программу:**
    ```bash
    poetry run python main.py
    ```
